# Ecom Ad Creative — Estrategia de Creativos de Alto Rendimiento

Esta skill proporciona conocimiento especializado para crear creativos publicitarios que detengan el scroll, generen clics de alta calidad y conviertan.

## Principios Fundamentales

### 1. La Regla del Hook
El **80% del rendimiento** de un creativo se decide en los primeros **3 segundos**. Si el hook no detiene el scroll, nada más importa.

**Métricas objetivo:**
- **Hook Rate > 30%** (% de personas que ven más de 3s)
- **CTR > 2%** (Click-Through Rate)
- **ThruPlay > 15%** (% que ven el video completo)

### 2. Los 4 Pilares de un Creativo Ganador
1. **Interrupción** — Detener el scroll (hook visual/auditivo)
2. **Conexión** — "Esto es para MÍ" (validar el dolor del avatar)
3. **Demostración** — Mostrar que funciona (prueba visual)
4. **Acción** — Decirle exactamente qué hacer (CTA claro)

### 3. Psicología del Scroll
- El cerebro procesa imágenes en **13 milisegundos**
- El contraste visual es lo primero que se detecta
- Las caras humanas reciben **2x más atención** que objetos
- El movimiento inesperado detiene el scroll automáticamente
- Los números concretos generan más engagement que texto genérico

## Frameworks de Hooks

### Framework PAVE (Para hooks de video)
- **P**rovocación — Desafiar una creencia ("Estás lavando tu carro MAL")
- **A**sombro — Resultado visual sorprendente ("Mirá esta diferencia")
- **V**alidación — Prueba social ("3,800 personas ya lo usan")
- **E**mergencia — Urgencia real ("Últimas unidades esta semana")

### Framework 3C (Para hooks de imagen)
- **Contraste** — Visual dramático (antes/después, split screen)
- **Curiosidad** — Dato incompleto que obligue a leer ("El 89% no sabe que...")
- **Cifra** — Número concreto que represente valor ("Ahorra Q1,200/año")

## Reglas de Retención para Video

### Los 5 "Anti-Abandono"
1. **Corte cada 2-3 segundos** — Cambio de ángulo, zoom, o toma diferente
2. **Texto en pantalla siempre** — Refuerza el audio y retiene sin sonido
3. **Progresión de valor** — Cada segundo debe agregar info nueva
4. **Cliffhanger interno** — "Pero espera, eso no es todo..." a mitad del video
5. **Loop** — El final conecta con el inicio para reproducción automática

### Curva de Atención ideal

## Prompts de IA — Mejores Prácticas

### Para imágenes de producto (Midjourney/DALL-E/Flux)
```
Estructura: [Sujeto] + [Acción] + [Contexto] + [Estilo] + [Iluminación] + [Cámara]

Ejemplo:
"Close-up photograph of a person applying a cleaning solution on a car windshield,
half of the windshield is foggy and oily, the other half is crystal clear and reflective,
dramatic before-and-after split, natural outdoor lighting, shot on Canon 5D Mark IV,
commercial product photography style, clean background"
```

### Para imágenes de estilo lifestyle
```
Estructura: [Persona] + [Emoción] + [Usando producto] + [Entorno] + [Estilo]

Ejemplo:
"Happy Latin American man in his 30s admiring his clean car windshield in a home garage,
holding a small bottle of cleaning product, warm morning light, suburban setting,
authentic candid photography, no text, no logos"
```

### Para imágenes de anuncios
```
Estructura: [Producto] + [Composición] + [Elementos de oferta] + [Estilo]

IMPORTANTE: El texto se agrega DESPUÉS en Canva/editor.
Solo generar la imagen base sin texto.
```

### Palabras clave por estilo:
| Estilo | Palabras clave en prompt |
|---|---|
| Producto limpio | "product photography, white background, studio lighting, clean, minimal" |
| Lifestyle | "candid, authentic, natural light, real person, everyday setting" |
| Dramático | "high contrast, dramatic lighting, before-after, split composition" |
| Infográfico | "flat design, clean infographic, icons, data visualization, modern" |
| Premium | "luxury, sleek, dark background, golden accents, high-end feel" |

## Estrategia de Testing A/B

### Nivel 1: Test de Hooks (Prioridad máxima)
- Mismo body + CTA, cambiar solo el hook
- Testear 3-5 hooks diferentes
- Ganador = el que tiene mayor Hook Rate

### Nivel 2: Test de Formatos
- Mismo mensaje, diferente formato (imagen vs video vs carrusel)
- Ganador = el que tiene mejor CTR

### Nivel 3: Test de Copy
- Mismo creativo visual, diferente texto (primary text, headline)
- Ganador = el que tiene mejor CPC

### Nivel 4: Test de Audiencias
- Mismo creativo ganador, diferentes segmentos
- Ganador = el que tiene mejor ROAS

**Regla:** Siempre testear UNA variable a la vez. Nunca cambiar hook + copy + imagen simultáneamente.

## Errores Comunes a Evitar

| Error | Por qué falla | Solución |
|---|---|---|
| Hook genérico | No detiene el scroll | Usar framework PAVE/3C |
| Solo producto, sin humano | Baja conexión emocional | Incluir elemento humano siempre |
| Video estático | Pierden atención en 2s | Cortes cada 2-3 segundos |
| CTA sin método de pago | Baja conversión | Siempre "Pago al recibir" |
| Promesa ≠ Landing | Alta tasa de rebote | Verificar coherencia antes de lanzar |
| Texto ilegible en móvil | No se lee el beneficio | Mínimo 48px en 1080px |
| Sin subtítulos | 80% ve sin sonido | Subtítulos siempre en video |
| Mucho texto en imagen | Meta penaliza > 20% texto | Máximo 20% del área con texto |
